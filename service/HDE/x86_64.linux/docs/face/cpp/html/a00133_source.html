18 API documentation {#mainpage_api}
19 =================
20 
21 The namespace FACE::TS contains all of the major functionality of the ISO C++ 2 FACE API.
22 
23 - FACE::TS provides facilities for:
24  - Initializing, Creating and Destroying a connection
25  - Sending and Receiving data
26  - Registering and Unregistering a callback.
27  - Getting Connection Parameters

29 DataType generation:
30 --------------------
31 Idlpp can be used to generate the CPP datatypes for the FACE API. The option -F can be used to generate the FACE specific datatypes.
32 
33 
34 Example command for code generation from the idl Foo.idl for Isocpp2:
35 
36 `idlpp -S -l isocpp2 -F Foo.idl`
