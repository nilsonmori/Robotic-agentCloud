11 Known Issues and Currently Unsupported Features {#release_notes_known_issues}
12 =====================================
13 
14 Although the ISO C++ DCPS API implements the majority of commonly used features, some functionality
15 of the OMG DDS specification and DDS-PSM-Cxx is not currently supported. Below is a list of known
16 issues and features that are not currently supported:

30 OpenSplice API extensions {#release_notes_us_extensions}
31 -------------------------
32 The implementation contains only API features that are present in the OMG DCPS specification at this time.
33 Other OpenSplice language bindings contain extensions to the specification such as additional QoS,
34 Entity operations, and Entities. These features are not yet available.
